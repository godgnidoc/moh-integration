# Moh 

墨（Moh）项目的目标是构建一套可复用可定制的机器人开发平台，基于此平台设计开发一系列满足民间竞技娱乐目的的机器人系统。

- 墨追求资产的可复用性和一致性，希望提高整个系统的参数化设计和定制能力
- 墨追求轻量和小巧，不论是目标产品形态还是迭代中的设计材料

# 里程碑1 第一人称视角云台 M.G1

第一人称视角云台作为墨的第一个里程碑，目标是探索完成一个跨越机械电子和软件领域的综合项目所需的项目结构和运作模式。以及初步学习完成各领域开发所需的基本技能。

- M.G1 由底座和头部构成
- 头部与底座之间由偏航和俯仰两部电机连接，可以执行偏航和俯仰动作
- 头部搭载一颗MIPS相机提供主要视觉输入，以及一些次要传感器和执行器
- 头部内置一块基于 RK3566 的开发板作为主控，与各传感器和执行器通信
- 电机由单独的驱动板控制，驱动板内置角霍尔传感器和驱动芯片，接收主控串口指令
- M.G1 通过一张电源适配板从 USB TypeC 接口获取 12v 直流电源
- M.G1 通过WiFi与PC连接，使用键鼠操控行动，将视觉输入以 H.254 传输至客户端

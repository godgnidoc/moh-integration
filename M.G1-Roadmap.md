# 客户端开发

开发 Win/Linux 下的图形应用程序，实现以下功能：

- 展示图形交互界面
- 允许用户搜索并选择在线设备
- 连接设备后与设备通信，展示设备回传图像界面
- 接收用户鼠标操作，转换为云台姿态指令传输给设备

# 设备端应用开发

在 RK3566 Linux 平台开发设备端应用，实现以下功能：

- 广播自身存在性
- 接受客户端连接（单点登录）
- 周期向客户端回传传感器数据
- 接收客户端回传指令
- 根据指令与执行器通信

# 电机驱动器程序开发

在 STM32F031C6T6 平台开发应用，实现以下功能：

- 周期读取角度传感器
- 周期上报执行器状态
- 持续计算目标姿态所需波形，输出 PWM 控制电机驱动电路

# 电机驱动器电路设计

适配 HT3505 电机设计带霍尔传感器的FOC驱动板

- 支持 AS5600 霍尔传感器
- 支持 DRV8313PWPR/SS6343M 电机驱动芯片
- 使用 STM32F031C6T6 主控
- 使用串口与主板通信，预留 SWD 调试线用于下载程序
- 12V直流电源输入为电机驱动芯片供电
- 使用 SSP7903P33PR 稳压至 3.3V 为其他芯片供电
- 提供适当的 LED 工作指示灯

# 电源适配器电路设计

- 支持 USB TypeC 接口电源输入
- 遵循PD协议请求 12V 电压输入
- 将 12V 直流电源分发给主板和电机驱动板

# 底软开发
